# Events Booking and Ticketing App

## Software Development 2 Coursework

The objective is to demonstrate the use of Object-Oriented Programming and Event-Driven Programming by integrating a database, graphical user interface and API usage.
This architecture implements the MVC design pattern by separating the concerns of each component.  Asynchronous programming was used throughout the project, from the user interface down to the database queries.

 - Package manager: uv
 - Backend: FastAPI
 - Database: MySQL
 - Frontend: HTML, CSS, JS
